==========================================
 Description
==========================================
VidLister shows your (youtube) videos on your site.
Thumbs are shown like a gallery and videos open inside a
lightbox.

==========================================
 Features
==========================================
* reads your Youtube feed and imports the video data
* shows all video thumbs (paginated) with link to video
* Languages (backend):
  - english
  - dutch

==========================================
 Requirements
==========================================
* MODX Revolution (tested with 2.1.3)
* jQuery for the lightbox

==========================================
 Installation
==========================================
* Install through Package Management
